# ip 地址管理器

因要修改学校网络的关系，我这里需要整一个自动适配学校网络的 东西


# 功能说明

![image-20211009082229601](https://raw.githubusercontent.com/m1n9yu3/image_markdown/main/blogimg/202110090822671.png)



1. 首先 当前ip 地址和 网关都会显示在最上方  并且实时刷新
2. 规则以文件的形式保存在 配置文件中
3. 点击加号减号时，会增加或删除规则
4. 点击规则时可以编辑规则
5. 规则是多条命令的组合形式
